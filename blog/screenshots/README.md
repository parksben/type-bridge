# 截图说明

本文档列出博客文章所需的截图清单，供截图 agent 执行。

---

## 通用说明

- 截图时 APP 应处于**已启动 WebChat 会话**的状态（桌面端有二维码，手机端可连接）
- 手机端页面通过**同 WiFi 下手机扫码桌面二维码**访问，或用浏览器模拟手机访问
- 所有截图放入 `blog/screenshots/` 目录

---

## 图1：APP 主界面截图

| 项目 | 说明 |
|------|------|
| **文件名** | `1-app-main-window.png` |
| **页面** | TypeBridge 桌面端主窗口 |
| **状态** | 已启动 WebChat 会话，展示二维码 + WiFi 名称 + 进度环 |
| **尺寸建议** | 约 1200×800 或等比裁切 |
| **截图内容** | 左侧边栏（连接手机 tab 高亮）+ 右侧 WebChat Tab 内容区（二维码、WiFi 名称、会话进度环、已绑定设备列表） |

> **怎么截**：打开 TypeBridge → 进入「连接 TypeBridge」→ 点「WebChat」→ 点「启动会话」，截取完整窗口（或仅内容区域）。

## 图2：手机端 WebChat 聊天页面

| 项目 | 说明 |
|------|------|
| **文件名** | `2-mobile-webchat-chat.png` |
| **页面** | 手机浏览器中的 WebChat 聊天页面 |
| **状态** | 已连接并完成握手，当前在「打字」Tab，有至少一条已发送并已注入成功的消息 |
| **尺寸建议** | 390×844（iPhone 14/15 比例） |
| **截图内容** | 顶部 Tab 栏（打字 / 触控 / 键盘）、消息气泡（含"已注入"状态徽标）、底部输入框和发送按钮 |

> **怎么截**：手机扫码连接后，在「打字」Tab 发送一条文字消息（如"你好 TypeBridge"），待消息气泡出现"已注入"标识后截图。
