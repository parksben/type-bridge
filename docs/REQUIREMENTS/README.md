## 文档约定

- REQUIREMENTS 记"做什么、为什么"——用户可见行为与产品决策
- 技术实现细节见 `docs/TECH_DESIGN/README.md`
- 改动影响用户可见行为 → 两份文档都更新；仅影响内部实现 → 只更 TECH_DESIGN
